# exercicio01-Devops

## SRE o que é?

Principalmente garantir que o sistema seja escalável, confiavel e eficiente, ele gasta menos tempo em tarefas manuais e mais em autonomas.

## Devops o que é?

Metodologia que tem como objetivo aumentar a velocidade de entrega e a qualidade ou seja faz o desenvolvimento e a operação trabalharem juntamente, em todo o ciclo de vida do software.
